HTML And CSS

Inline: This is used when the user wants to change a object on one page with out changing similar objects of other pages for example if a person wanted to change a heading on a page to a different colour, but using inline CSS this is how could do it. However by using inline it can be very in efficient as this may need to be repeated which means that inline can be very time consuming to use. Internal: This type of CSS styling is written within the header of a webpage, below the title tags and placed within style tags so the browser knows that that is the CSS code needed, by using inline CSS styling this ad the clear advantage over inline as there will be less repeated coding when wanting to change an item which also means it’s a time saving measure, although this will still have to be repeated on every wag that the website has. External: The most common type of styling, External CSS uses style sheets that are created and located within the same folder in which the website is stored in to. These styles will be applied to all of the pages that are between the ‘html‘ tags. Which is also its main advantage, the fact that it is applied to all of the sites webpages which reduces unnecessary lines of code and save the programmer time.
